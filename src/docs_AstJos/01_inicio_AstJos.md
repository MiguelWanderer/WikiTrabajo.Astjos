# Inicio de la Wiki AstJos

## Introducción

Esta wiki reúne los conocimientos esenciales para servir a la Unidad 2 del curso de Sistemas Operativos. Cada sección está diseñada para crear un paso a paso ordenado de todos nuestros procesos, asegurando la correcta instalación y funcionamiento de Windows Server.

## Objetivo

Servir como guía central completa para la instalación, configuración y operación del entorno de laboratorio de Windows Server 2025, con enfoque especial en:
- **Active Directory** - Gestión centralizada de usuarios y recursos
- **Cliente de dominio** - Integración de equipos en el dominio
- **Servicios de red** - Configuración de servicios fundamentales
- **Políticas de grupo (GPO)** - Control de configuración en la red

## Temas Principales

Este laboratorio se divide en los siguientes módulos:

1. **Instalación y Preparación (Paso A)** - Configuración inicial del servidor
2. **Active Directory (Paso B)** - Instalación y configuración de AD
3. **Cliente de Dominio (Paso C)** - Integración de clientes en el dominio
4. **Servicios de Red (Paso D)** - DHCP, DNS y otros servicios
5. **Políticas de Grupo (Paso E)** - Implementación de GPOs
6. **Configuración Avanzada (Paso F)** - Optimización y seguridad

## Requisitos Previos

- Conocimientos básicos de sistemas operativos
- Acceso a VirtualBox o similar
- ISO de Windows Server 2025
- Mínimo 50 GB de espacio disponible
- Conexión a internet

## Cómo Usar Esta Wiki

- Cada sección incluye pasos detallados y referencias visuales
- Sigue los pasos en orden para máxima efectividad
- Toma snapshots después de cada configuración importante
- Documenta cualquier desviación del procedimiento
