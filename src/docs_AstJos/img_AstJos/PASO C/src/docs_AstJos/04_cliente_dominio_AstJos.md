# Cliente de dominio

En esta sección se documenta la creación de la OU, los usuarios y el grupo que se utilizarán dentro del dominio inacap.local para continuar con el proceso de laboratorio.

## Creación de OU, usuarios y grupo

Abrir Usuarios y equipos de Active Directory desde Administrador del servidor, entrando en Herramientas y seleccionando esa consola.

Crear la OU Ventas llamada J-ventas. Para ello, tenemos que hacer click derecho sobre inacap.local, luego ve a Nuevo, elige Unidad organizativa y asigna el nombre Ventas.

Crear los usuarios dentro de la OU Ventas. Haz clic derecho sobre la OU, entra a Nuevo y después a Usuario.

Crear 2 usuarios; uno con tu código personal, por ejemplo ATSJOS.

Definir la contraseña para cada cuenta, usando la misma que hemos utilizado para todo el proceso.

Para el laboratorio, desmarcar el usuario debe cambiar la contraseña en el próximo inicio.

Crear el grupo J-Ventas dentro de la OU Ventas. Haz clic derecho sobre Ventas, luego Nuevo, después Grupo y asigna el nombre J-Ventas.

Agregar los usuarios al grupo desde cada cuenta, entrando a la pestaña Miembro de, usando Agregar e incorporándolos al grupo J-Ventas.
